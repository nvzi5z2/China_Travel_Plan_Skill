# travel-planner — 旅游规划 Skill

> 基于 Claude Code 的 AI 旅游规划助手，从需求收集到生成手机端 HTML 攻略，全流程自动化。

## 工作流程（5阶段）

1. **需求收集** — 一次一个问题，问清出发地、航班、酒店、偏好、必去项
2. **信息调研** — 并行搜索天气/航班/美食（大众点评评分）/景点/当地活动
3. **MD攻略生成** — 管家语气，含行李清单、每日时间线、美食推荐
4. **HTML界面生成** — 移动优先单页应用，底部Tab导航，折叠卡片
5. **顾问审计** — 自动检查预算、备选方案、拍照点、方言、浪漫时刻

## 文件结构

```
.claude/skills/travel-planner/
├── SKILL.md        # 5阶段流程定义 + 设计规范 + 反模式清单
└── template.html   # CSS设计系统 + 组件骨架（参数化配色）
```

## 使用方式

1. 将 `.claude/skills/travel-planner/` 复制到你的项目目录下
2. 在 Claude Code 中说"帮我规划一个XX旅游"
3. Skill 自动加载，从提问开始走全流程

## 输出示例

- 完整的 Markdown 旅行攻略（行程 + 美食 + 贴士）
- 手机友好的单页 HTML 界面（底部Tab导航、折叠卡片、餐厅评分徽章、行李勾选清单）

## Skill 设计原则

- **专属旅游管家**语气，非通用模板
- 每类餐厅推荐3家，附大众点评评分+人均+距离
- 行李清单按人头分：共用/你的/她的
- 社交媒体热推品类优先（小红书/抖音/B站）
- 自动审计：花费预估、雨天备选、拍照点、方言速成、浪漫时刻
