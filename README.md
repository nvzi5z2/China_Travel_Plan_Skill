# travel-planner — AI 旅游规划 Skill

> 基于 Claude Code 的 AI 旅游规划助手，6 阶段流水线，集成飞猪 flyai 实时搜索机票/酒店/租车。

## 工作流程

0. **用户画像** — 8 维度画像（出行组合/饮食/体力/节奏/花钱风格/特殊日子/雷区/兴趣）
1. **深度需求收集** — 基于画像的交通/住宿/租车偏好（不预设推荐结论）
2. **双通道调研** — Web搜索（天气/美食热度/景点）+ flyai MCP实时查询
3. **LLM推理推荐 + MD攻略** — 结合画像+flyai数据逐个评估，每条推荐附「为什么」
4. **HTML界面** — 移动优先单页，含交通/酒店/租车预订卡片
5. **管家审计** — 14项检查（含美食覆盖率/推荐自洽性/体力曲线）

## 文件结构


## 设计原则

- **LLM做判断，脚本做数据** — 脚本只做数据清洗，推荐由LLM推理产出
- **画像是推理上下文，不是查找表** — 不硬编码推荐，LLM逐个评估
- **美食嵌入行程** — 每天时间线含餐厅+评分+推荐菜+人均，不独立成章
- **全中文指令** — 指令语言=执行语言=输出语言
- **反幻觉优先** — 每条事实有搜索来源，flyai数据标注查询时间
- **可预订** — 交通/酒店卡片集成飞猪商品链接

## 使用方式

1. 将 `.claude/skills/` 复制到项目目录
2. 安装 flyai CLI：`npm i -g @fly-ai/flyai-cli`
3. 在 Claude Code 中说「帮我规划一个XX旅游」自动触发

## 依赖

- Python 3（脚本）· Node.js（flyai-cli）· `npm i -g @fly-ai/flyai-cli`
